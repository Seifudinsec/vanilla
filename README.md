# Vanilla — Pure. Simple. Extraordinary.

Artisan coffee single-page website with full GSAP animation suite. Monogram logo: **VN**.

## Stack

HTML5 + CSS3 + Vanilla JS + GSAP 3.12.5 (ScrollTrigger, ScrollToPlugin) + Instrument Serif / Inter

## Run

```bash
npx serve . -p 3000 -s --no-clipboard
```

## GSAP Features

- Loader with VN logo rotation and reveal timeline
- Split-text stagger on "VANILLA" hero headline
- 20 floating steam particles with random paths
- Mouse-follower copper glow
- Story: 3-panel horizontal scroll-track, pinned + scrubbed with progress bar
- Menu: horizontal card scroll with containerAnimation
- Gallery: 3D perspective tilt on hover, blur-to-focus scroll reveals
- Contact: scroll-revealed info, pulsing map marker
- Nav: transparent-to-solid, active highlight, smooth scroll, mobile slide-in
- Full `prefers-reduced-motion` support

## VN Logo

Interlocking geometric monogram — V and N share a continuous stroke path. Rendered as inline SVG throughout (loader, nav, footer).
